# Integrations

nexus-fs integrates with the Python data science and AI agent
ecosystems through standard interfaces.

| Integration | What it gives you |
|-------------|-------------------|
| [fsspec](fsspec.md) | Use nexus-fs as a registered filesystem in any fsspec-compatible library |
| [Data Science](data-science.md) | pandas, dask, and HuggingFace examples using the fsspec adapter |
| [Agent Frameworks](agent-frameworks.md) | LangChain, CrewAI, Claude SDK, OpenAI Agents, and LangGraph |
